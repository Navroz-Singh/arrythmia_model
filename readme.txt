How to Run the Project ->

Download Data: (If not done already)
python download_data.py

Preprocess Data: This runs the new, advanced preprocessing pipeline.
python data_preprocessing.py

Train Model: This will train the advanced ResNet model.
python train.py

Evaluate Model: This provides a comprehensive evaluation of the trained model.
python evaluate.py

Make a Prediction: Run this to see how the model performs on a single, specific heartbeat.
python predict.py
